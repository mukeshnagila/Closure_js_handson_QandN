Q.1 - 
    function counter(){
    var counter = 0;
    function IncreaseCounter(){
        return counter += 1;
    };
    return IncreaseCounter;
    }
    var counter = counter();
    alert(counter());
    alert(counter());
    alert(counter());
    alert(counter());"

Ans -

here is the answer -- The output is on alert 1,2,3 and 4..


Explain---    The code defines a function called "counter" that returns an inner function "IncreaseCounter". This inner function increments a variable counter each time it is called.
When the counter function is invoked and assigned to a variable, it creates a closure, preserving the reference to the "counter" variable.
Subsequent calls to the variable (which now holds the "IncreaseCounter" function) increment and return the updated value of the "counter" variable.
The provided code displays a series of 'alerts', showing the incremented values of the counter: 1, 2, 3, and 4.

  var a = 12;
  return function () {
    alert(a);
  };
})();
x();

Ans -

The output is 12... You can check this ouput on alert...
click the button and check...


Explain---    this code, two variables are declared: 'a' and 'x'. The global variable a is assigned the value '10'. Then, an IIFE is used to assign a function to the variable 'x'. Inside the IIFE, there is a local variable a with a value of '12', which shadows the global variable a. The IIFE returns an inner function that accesses the local a and displays its value using the "alert()" function.
When the x function is invoked with x(), it executes the inner function defined inside the IIFE, and the 'alert(a)' statement displays the value of the local variable a, which is 12......
